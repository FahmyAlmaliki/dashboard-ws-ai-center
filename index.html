Tentang Workshop AI Center

Membangun Keterampilan AI Melalui Project Nyata

Program Workshop Kami

Workshop AI Center UB adalah sebuah organisasi yang berfokus pada pengembangan kompetensi kecerdasan buatan melalui kegiatan pelatihan, riset, dan kolaborasi. Setiap anggota didorong untuk menghasilkan project nyata yang dapat diimplementasikan dan memberi manfaat bagi masyarakat

Fokus Workshop:

  • Pengembangan AI Applications
  • Machine Learning Implementation
  • Internet of Things Projects
  • Computer Vision Solutions
  • AI Tools & Platform Integration
  • Deployment & Production Ready Apps

Project Workshop Kami

Hasil Karya Peserta Workshop AI Center

Weather Jombang

Aplikasi prediksi cuaca untuk wilayah Jombang menggunakan machine learning dan data real-time dari berbagai sumber.

Machine Learning Weather API Web App

Chatbot AI Assistant

Intelligent chatbot menggunakan NLP untuk menjawab pertanyaan akademik dan layanan mahasiswa UB.

NLP ChatGPT API Python

Image Recognition App

Aplikasi pengenalan objek menggunakan deep learning untuk identifikasi tanaman dan hama pertanian.

Computer Vision TensorFlow Mobile App

Predictive Analytics Dashboard

Dashboard analitik prediktif untuk monitoring dan forecasting data akademik universitas.

Data Science Analytics Dashboard

AI Translator

Sistem penerjemah otomatis Bahasa Indonesia - Bahasa Daerah menggunakan neural machine translation.

NMT Transformer API

Video Summarizer

Tools AI untuk membuat ringkasan otomatis dari video pembelajaran menggunakan speech-to-text dan NLP.

Speech AI NLP Video Processing

Document Analyzer

Sistem analisis dan ekstraksi informasi dari dokumen menggunakan OCR dan NLP untuk otomasi administrasi.

OCR Document AI Automation

Recommendation System

Sistem rekomendasi mata kuliah dan karir untuk mahasiswa berdasarkan minat dan performa akademik.

Recommendation ML Personalization
