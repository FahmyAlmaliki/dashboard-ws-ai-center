# Dashboard AI Center Universitas Brawijaya

Website dashboard untuk organisasi AI Center Universitas Brawijaya.

## Fitur

- **Desain Modern & Responsif** - Tampilan yang menarik dan mobile-friendly
- **Navigasi Smooth** - Navigasi yang halus antar section
- **Counter Animation** - Animasi counter untuk statistik
- **Project Showcase** - Menampilkan project-project unggulan
- **Team Section** - Profil tim AI Center
- **Contact Form** - Form kontak interaktif
- **Jombang Portal Button** - Tombol akses cepat ke portal Jombang (devel-ai.ub.ac.id/jombang)

## Teknologi

- HTML5
- CSS3 (dengan Custom Properties/Variables)
- Vanilla JavaScript
- Font Awesome Icons

## Cara Menggunakan

1. Buka file `index.html` di browser Anda
2. Website akan langsung berfungsi tanpa perlu instalasi tambahan
3. Untuk deployment, upload semua file ke web server Anda

## Struktur File

```
dashboard-AI-Center/
├── index.html      # File HTML utama
├── styles.css      # File styling CSS
├── script.js       # File JavaScript untuk interaktivitas
└── README.md       # Dokumentasi
```

## Fitur Khusus

### Button Jombang Portal
Terdapat button khusus di navbar dan hero section untuk mengakses portal Jombang:
- URL: https://devel-ai.ub.ac.id/jombang
- Membuka di tab baru
- Styling yang menarik perhatian

### Sections

1. **Hero** - Bagian pembuka dengan CTA button
2. **Stats** - Statistik organisasi dengan animasi counter
3. **About** - Informasi tentang AI Center
4. **Projects** - Showcase project-project unggulan
5. **Team** - Tim dan peneliti
6. **Contact** - Form kontak dan informasi kontak

## Customization

Anda dapat mengkustomisasi website dengan mengubah:

- **Warna**: Edit CSS variables di `:root` pada `styles.css`
- **Konten**: Edit teks di `index.html`
- **Logo**: Ganti URL logo di tag `<img>` navbar
- **Informasi Tim**: Edit bagian team section
- **Projects**: Tambah/edit card di bagian projects

## Browser Support

Website ini mendukung browser modern:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
